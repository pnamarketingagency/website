HOW TO USE THIS GUIDE
----------------------
1. Find the image filename in the relevant section below.
2. Source or shoot a photo matching the description.
3. Resize and compress it using squoosh.app or tinyjpg.com.
4. Drop the file into the same folder as your HTML files.
5. The HTML already references these filenames — it will just work.

For images currently using Unsplash placeholder URLs, find the
HTML comment directly above the <img> tag. It looks like this:
    <!-- IMAGE: filename.jpg ... -->
Replace the src="https://images.unsplash.com/..." with your file.

GENERAL TIPS
================================================================

COMPRESSION TOOLS
-----------------
• squoosh.app      — Best quality/size control. Use WebP for modern browsers.
• tinyjpg.com      — Quick drag-and-drop compression for JPG and PNG.
• imageoptim.com   — Mac app for batch compression.

FORMATS AT A GLANCE
-------------------
• JPG  — photos, hero images, cards. Set quality 75–85%.
• PNG  — logos, icons, anything needing transparency.
• WebP — best modern format. Smaller files, same quality.
         Use with a JPG fallback for older browsers.
• SVG  — ideal for logos and icons. Resolution-independent.

SIZING GUIDELINES
-----------------
• Never upload images larger than needed.
• Hero images: 900px wide is plenty for most screens.
• Service card images: 800px wide max.
• Logo: any size, will be scaled by CSS.
• Always provide a descriptive alt attribute for accessibility.

PHOTOGRAPHY STYLE FOR MARKETING AGENCY
---------------------------------------
• Natural light wherever possible — avoid harsh flash.
• Warm, professional tones that feel real, not stock.
• People working, collaborating, thinking — action beats pose.
• Clean, uncluttered backgrounds add to the professional feel.
• Phones, laptops, and monitors showing real content add credibility.
• Australian settings add local trust — think QLD architecture,
  recognisable spaces, or subtle local cues.
